## What This Is

Minecraft Old-School brings back the Beta 1.3 feel through a fully web-ready build based on Eaglercraft. It keeps the early Minecraft mood intact while layering in modded additions that broaden the experience. You can jump into the familiar block-building and survival loop right in your browser, with no installation step needed.

The aim of the project is to preserve the original Beta 1.3 style while introducing custom changes that add depth without losing the retro identity. If you're returning for nostalgia or trying classic Minecraft for the first time, this web implementation makes it easy to play. An offline download is also available so the game can be kept for use without an internet connection.

## FAQ

**How do I begin?**  
Open the downloaded HTML file or visit the web page. The game loads directly in your browser.

**What happens to saved worlds after updates?**  
World saves live in your browser's local storage. If you clear browser cache, your saved progress may be removed.

**Can I change the mod myself?**  
Yes. It is open source under GPL v3.0, so you can edit the HTML and JavaScript files to add or remove features.

**Does it work with standard Minecraft?**  
No. This is a separate web-based implementation and does not connect to official Minecraft installations or servers.

**How do I use it offline?**  
Use the download link above. The full game is packaged in a single HTML file that functions without internet access.
